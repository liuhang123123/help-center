# STEP1 ：选择币种&下单模式

## 第一步：选择币种

在Bitrader主界面USDT本位永续合约列表中，选择需要交易的U本位合约币种。没有展示出来的币种可以通过点击左侧搜索按钮搜索。

## 第二步：选择下单模式

下单模式有三种：现价下单、市价下单、计划订单，可根据需要选择下单模式。
