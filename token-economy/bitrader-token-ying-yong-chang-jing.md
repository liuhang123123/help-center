## Bitrader Token使用场景

用户使用Bitrader App交易（通过API连接OKX交易所）的同时，可赚取到额外“使用”收益。

{% hint style="info" %}
奖励以Bitrader Token的形式进行发放，包括“邀请奖励、交易奖励（包括自己和邀请的人）、节点奖励（如果是节点）、平台奖励”。
{% endhint %}

## Bitrader Token价值来源

Bitrader平台90%的收益将用于回购Bitrader Token。

## Bitrader Token锁仓权益

Bitrader致力于向用户提供高实用性工具，未来将有更多高级功能逐渐向用户解锁。
